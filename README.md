# SYNERGEN: High-Precision Meritocratic Allocation Engine (v3.0)

**SYNERGEN** is an algorithmic framework designed to maximize systemic efficiency by matching human cognitive potential (Nodes) to critical global challenges through weighted trait-vector analysis.

## 🚀 Core Philosophy
Current resource allocation systems rely on static credentials. SYNERGEN introduces the **S-Coefficient**, a dynamic metric that evaluates the synergy between a candidate's cognitive fingerprint and a specific problem's requirements.

### The Mathematics of Synergy
The system's efficiency is governed by the formula:

$$S = \frac{\Delta U}{\sigma \cdot C}$$

Where:
* **$\Delta U$ (Utility Gain):** Total projected benefit or resource savings.
* **$\sigma$ (Risk Sigma):** Systemic risk factor (0.0 to 1.0).
* **$C$ (Cost):** Total implementation and resource expenditure.

---

## 🛠 System Architecture
The engine operates through a three-stage pipeline:

1.  **Cognitive Assessment:** Candidates provide qualitative responses to systemic problems.
2.  **Trait Vectorization:** Responses are analyzed via logic markers and keyword density to create a multi-dimensional trait profile.
3.  **Weighted Matching:** The engine projects the Trait Vector against a task's weighted requirements (`weights.json`) to calculate the Final Match Score.

### 📂 File Structure
* `main.py`: The system entry point (v3.0).
* `core_engine.py`: Handles the mathematical S-Coefficient logic.
* `cognitive_assessment.py`: Transforms text into numerical trait data.
* `matching_engine.py`: Executes the matching algorithm.
* `active_challenges.py`: Manages the database of systemic tasks.
